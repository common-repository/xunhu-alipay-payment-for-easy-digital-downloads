== Installation ==
1. 先安装EDD插件
2. 安装 "xunhu-wechat-payment-for-Easy Digital Downloads" 插件并激活
3. <a href="https://admin.xunhupay.com">注册账号</a>
4. 申请支付宝收款通道
5. 创建网站应用，获取APPID和secret
6. 在插件设置中填入APPID和secret
