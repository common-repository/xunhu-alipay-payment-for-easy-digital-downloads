== Installation ==
1. Install Easy Digital Downloads plugin and configure your store (if you haven't done so already - https://wordpress.org/plugins/easy-digital-downloads/).
2. Install "xunhu-Alipay-payment-for-Easy Digital Downloads" wordpress plugin just like any other Wordpress plugin.
3. Activate.
4. Bound to personal Alipay account
5. bulid a new application get appid and key put in plugin 
6. If the main currency of your store is not Chinese Yuan, please also set the exchange rate so that Alipay can convert the price to Chinese Yuan.
